# Security Policy

## Reporting a Vulnerability

Please report security issues privately using GitHub Security Advisories:

https://github.com/cbritt0n/greengate/security/advisories/new

Include a detailed description, reproduction steps, logs, and impact assessment. We will acknowledge receipt within **48 hours** and provide a remediation timeline within **5 business days**.

Do **not** disclose vulnerabilities in public issues, pull requests, or social media until a fix has shipped and a coordinated disclosure date has passed.

## Supported Versions

| Version | Supported |
| --- | --- |
| `main` / latest release | ✅ |
| Older tags | ❌ |

## Responsible Disclosure Process

1. Report privately via email.
2. Work with maintainers on reproduction + fix.
3. Receive CVE or advisory ID (if applicable).
4. Public disclosure coordinated after patch availability.

Thank you for helping keep GreenGate secure and trustworthy.
